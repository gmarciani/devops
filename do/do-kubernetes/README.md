# DO KUBERNETES

*Provisioning of a Kubernetes cluster on Digital Ocean with Ansible*


## Requirements
Install the plugin *vagrant-digitalocean*:

  $> vagrant plugin install vagrant-digitalocean


## Deploy
Create droplets:

  $> bash init.sh

Destroy droplets:

  $> bash destroy.sh

Provision droplets:

  $> bash provision.sh
